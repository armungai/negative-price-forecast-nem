# NEM Negative Price Forcasting

## Quick Start
### 1. Need a GitHub personal Access Token
    1. Go to github.com/settings/tokens
    2. Click *Generate new token (classic)*
    3. give it a name
    4. Select the repo scope (full control of private repositories)
    5. Click *Generate Token*
    6. Copy the token (won't see it again)

### 2. Save the token as a Colab Secret
    1. Open a notebook in Google Colab (any)
    2. Click the key icon in the left sidebar (Secrets)
    3. Click "Add new secret"
    4. Name: `GITHUB_TOKEN`, Value: paste the token
    5. Toggle Notebook access on
